II. FLOW HOẠT ĐỘNG
===================

User Input
    ↓
App.main() → Parse arguments
    ↓
AnalyzerService.analyze() → Parse Java source
    ↓
ClassModel + MethodModel + ParameterModel (Extracted metadata)
    ↓
TestGenerator.generateTests() → Convert models
    ↓
JavaPoetTestGenerator → Generate JUnit test code
    ↓
Write .java files → target/generated-tests/

III. USAGE EXAMPLES
===================

1. Sinh test cho single file:
   mvn exec:java -Dexec.mainClass="com.doantn.app.App" -Dexec.args="path/to/MyClass.java"

2. Sinh test cho directory:
   mvn exec:java -Dexec.mainClass="com.doantn.app.App" -Dexec.args="src/main/java target/my-tests"

3. Chạy script (Windows):
   run_test_generation.bat

4. Chạy script (Linux):
   bash run_test_generation.sh
