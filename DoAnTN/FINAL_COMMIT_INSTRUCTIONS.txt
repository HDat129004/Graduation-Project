================================================================================
                      TROUBLESHOOTING
================================================================================

Problem: "fatal: not a git repository"
Solution: Run from project directory
  cd C:\Users\Admin\Desktop\DATN\DoAnTN
  Then run the script again

Problem: "Authentication failed"
Solution: Use GitHub personal access token instead of password
  1. Get token from: https://github.com/settings/tokens
  2. Use token as password when prompted

Problem: "fatal: remote origin already exists"
Solution: Update the remote URL
  git remote set-url origin https://github.com/HDat129004/Graduation-Project.git
  Then push: git push -u origin main

Problem: "could not read Username"
Solution: Configure git user info
  git config --global user.name "Your Name"
  git config --global user.email "your@email.com"
  Then try again

Problem: "permission denied"
Solution: Check you have write access to the GitHub repo
  • Verify you own the repository
  • Check collaborator settings
  • Try using HTTPS instead of SSH
